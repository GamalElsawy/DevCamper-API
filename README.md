# DevCamper-API
Backend API for DevCamper application, which is a whole learning system includes bootcamps, courses, reviews and user authentication.

# How to use?
 - RUN npm install
 
# Run in development mode
 - npm run dev

# Run in production mode
 - npm start

# Destroy all data
 - node seeder -d

# Import all data
 - node seeder -i
